# TICKET-000: Add proper interactive tutorial

## Description
The current tutorial is only a poorly drawn hand showing how to draw a line. I want a clearer tutorial that looks like the rest of the design and that clearly shows the game play. Both how to generate fences, but also how the top menu works, as well as the bottom statistics. And that there is a store inbetween each map that allows for you to level up.

## Requirements
- [ ] Remove current tutorial
- [ ] Add a new tutorial for showing how to draw a fence. Only remove this part of the tutorial, once the user has drawn a fence.
- [ ] Add another step in the tutorial that highlights the top bar and its purpose
- [ ] Add another step in the tutorial that highlights the bottom bar and its purpose
- [ ] Add another step in the tutorial that informs the user about the store
- [ ] Once completed, the tutorial will not show again until you re-enable it from the options screen


## Technical Notes
- The tutorial needs to work as an overlay
- The tutorial, except for the first step, needs to have a clear next-button
- The tutorial should make it clear there is a Tutorial-section on the main screen

## Acceptance Criteria
- Users can understand the basic mechanics of the game
- Users can re-run the tutorial by re-enabling it from the options screen
